# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-13

### Added
- Initial release
- Core PluginUpdateChecker class
- PSR-4 autoloading support
- Automatic update checks with configurable intervals
- License key support for premium plugins
- WordPress plugin information popup integration
- Update cache system to reduce server load
- Force update check functionality
- Composer package support
- Complete documentation and examples
- Zero external dependencies

### Features
- WordPress-compatible API format
- Seamless integration with WordPress update system
- Support for plugin banners, icons, and screenshots
- Changelog display in update notifications
- Manual update check capability
- License key management
- Server-side validation support

### Security
- Uses WordPress core HTTP functions
- Follows WordPress coding standards
- No direct database queries
- Sanitized user inputs
- Secure license key transmission
